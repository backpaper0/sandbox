# JSF ( PrimeFaces )のサンプル

## これはなに

JSF自分勉強用のサンプルです。

コンポーネントは [PrimeFaces](http://primefaces.org/) を使用しています。

## 動かすには

Payara Microで動かすGradleタスクを書いています。
次のコマンドを実行してください。

```sh
gradle run
```

`control + c` で停止します。
