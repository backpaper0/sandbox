# LangGraphを試す

- [グラフ](./graph.md)の作成
  ```bash
  poetry run task graph
  ```
- ソースコードのフォーマット
  ```bash
  poetry run task format
  ```
- 静的解析
  ```bash
  poetry run task lint
  ```
- 自動修正
  ```bash
  poetry run task fix
  ```
- テスト実行
  ```bash
  poetry run task test
  ```
- 単純なチャット
  ```bash
  poetry run task chat -q "こんにちは！"
  ```
