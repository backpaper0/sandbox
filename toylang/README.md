# toylang

小さなプログラミング言語を作りたい。

とりあえずベタ（？）な感じで。

## 実装したい機能

- 整数、真偽値、文字列（、リスト、辞書）
    - `0` `1` `100` `-2`
    - `true` `false`
    - `"hello"` `""`
- 整数の四則演算
    - `+` `-` `*` `/`
- 整数の比較演算
    - `<` `<=` `>` `>=` `==` `!=`
- 真偽値の論理演算
    - `&&` `||`
- 文字列の結合演算
    - `+`
- 文字列の比較演算
    - `<` `<=` `>` `>=` `==` `!=`
- 変数
    - `var a = 100`
- 分岐、繰り返し
    - `if (a < 100) {} else {}`
    - `while (a < 100) {}`
- 関数定義、関数適用
    - `def foo(bar, baz) {}`
    - `foo("hello", 123);`

## 実装する順番の想定

- 抽象構文木
- 評価器
- 型推論器
- 型チェッカー
- パーサー
